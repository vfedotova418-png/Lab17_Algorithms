## Ответы на вопросы:

### 1. Что такое алгоритм?

**Алгоритм** — это точная конечная последовательность действий, направленная на
достижение определённой цели или решение конкретной задачи.

### 2. Как составляются блок-схемы?

У блок-схем в овальной схеме всегда есть "начало" и "конец". Построенние идёт обычно сверху вниз и слева направо. Для проверки условия используется ромб, для ввода/вывода используется параллелограмм, в прямоугольники помещаются операции/действия. Все части блок-схемы соединяют друг с другом стрелками.
